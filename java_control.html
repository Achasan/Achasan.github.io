[Java] 제어문(Control)

제어문(Control)

2021.08.02 ~ 2021.08.05

제어문

개요

코드는 위에서 아래로 순차적으로 진행된다. 이러한 흐름(flow)를 바꾸는 역할을 하는 문장들을 제어문(Control Statement)이라고 한다. 제어문은 크게 조건문과 반복문이 있는데 조건문은 조건에 따른 다른 문장이 수행되도록 하는 것, 반복문은 특정 문장들을 반복해서 수행한다.

조건문 : if

if문은 가장 기본적인 조건문으로, 조건식과 괄호{}로 이루어져 있다. 조건식이 참이면 괄호 내에 있는 구문을 실행하라는 의미를 가진다.

if(조건문) {
	//조건문이 true를 반환할 경우 실행할 실행문장을 적는다.
}

조건식

if문의 조건문에 사용 할 수 있는 연산자는 비교연산자와 논리연산자이다. 조건식은 boolean 타입의 true와 false를 반환해야하기 때문이다. 조건문은 무조건 true 또는 false를 반환해야한다.

블럭 {}

괄호 {}를 이용해서 여러 문장을 한 묶음으로 묶을 수 있는데 이를 블럭(block)이라고 명한다. 블럭은 괄호 내에 문장을 작성하기만 하면 된다. 단, 블럭은 끝 괄호 } 에 세미콜론(;)을 붙이지 않는다음 점을 알아야한다. if문의 실행문이 한 줄일경우에는 블럭을 사용안해도 문장이 실행되지만, 두 줄 이상 실행구문을 작성할 때에는 블럭을 작성해 주는 것이 코드의 가독성을 높일 수 있는 방법이다.

if - else 문

if-else 문은 if 조건문에 조건을 하나 더 추가하는 구문이다. if의 조건식이 참을 반환할 때, 실행문이 실행되는 것 처럼 조건식이 false를 출력할 때 실행할 구문을 else 블럭 생성을 통해서 지정할 수 있다.

if(조건식) {
		// 조건식이 true를 반환하면 실행될 실행문
} else {
		// 조건식이 false를 반환하면 실행될 실행문
}

if문을 2개 사용하는 것보다 상반된 결과를 출력한다면 if-else를 사용하는 것이 코드를 효율적으로 작성할 수 있어서 좋은 방법이 될 수 있다.

if-else if 문

if-else문은 두 가지의 경우 중 하나가 수행되는 구조이다. 하지만 경우의 수가 셋 이상인 경우에는 여러 개의 조건식을 쓸 수 있는 if-else 문을 사용하면 된다.

if(조건식1) {
		// 조건식1이 true를 반환하면 실행될 실행문
} else if (조건식2) {
		// 조건식1이 false이고 조건식2가 true를 반환하면 실행될 실행문
} else if (조건식3) {
		// 조건식1이 false, 조건식2가 false, 조건식3이 true를 반환하면 실행될 실행문
} else {
		// 3개의 조건식 모두 false를 반환할 때 실행될 else 문
}

마지막의 else 블럭은 생략이 가능하다. 하지만 보통 마지막에는 else블럭을 사용하여 구문을 종료한다. 조건문은 위에서부터 아래로 하나씩 맞는 조건이 있는지 확인한 후, 맞는 조건이 있는 조건식을 발견한다면 그 조건식에 맞는 else if문을 실행하고 조건문을 빠져나온다.

중첩 if 문

if문 안에 새로운 if문을 작성할 수 있다. 이를 중첩 if문이라고 한다. 중첩의 횟수에는 거의 제한이 없으며, 중첩 if문을 작성할 때에는 들여쓰기를 통해 범위를 명확히 구분해놓아야 코드를 읽을 때 혼동이 없다. 블럭생성시 사용하는 괄호{}를 활용하여 코드 실행 시 문제가 발생하지 않도록 하는 습관을 들이는 것이 좋다.

int score = 95;

	if(score >= 90) { // score가 90점 이상일 경우
		if(score >= 95) { // score가 90점 이상인 경우에 score가 95점 이상인지 확인
			System.out.println("A+ 학점입니다."); // true일 경우 실행문
		} else {
			System.out.println("A 학점입니다."); // false일 경우 실행문
		}
	} else { // score가 90점 이상이 아닐 경우
		System.out.println("점수가 90점 미만입니다."); // false일 경우 실행문
	}

다음의 코드에서는 score 변수를 95라고 선언하고, 조건문에 대입하였다. score가 90점 이상인 경우에는 95점 이상인지 한번 더 if문이 실행된다. 95점 이상일 경우에는 A+ 학점이라고 출력되도록 하였고, 90점 이상이지만 95점 이상은 아닐 시에는 A 학점이 출력되도록 작성하였다.

만약 score가 90점 이상이 아닐 경우에는 점수가 90점 미만이라는 출력을 실행하는 else 구문을 사용하여 조건문을 끝마치도록 작성하였다.

처음 중첩 if문을 작성할 때는 헷갈릴 수 있으나, 스무고개하듯이 조건을 이리저리 맞춰보면서 맞는 조건을 찾는다고 생각하는 것이 편하다. 무엇보다 조건문에 익숙해지는 것이 중요하다.

switch-case 문

switch-case 문은 처리할 경우의 수가 많은 경우에 간결한 조건문 작성을 할 수 있도록 해준다. 물론 제약조건이 있지만, 이러한 조건을 만족할 경우에는 if문을 여러 개 작성하는 것 보다 switch-case 문을 작성하는 것이 매우 효과적이다.

switch문은 조건식을 먼저 계산한 후에, 여러개의 case 문 중에 조건과 일치하는 case문으로 이동한다. 이동한 후에는 case문에 작성된 실행문들을 실행하고 끝에 break 문을 만나서 구문을 벗어나게 된다.

💡 1. 조건식을 계산한다. 2. 조건식의 결과와 일치하는 case문으로 이동 3. 이후의 문장들을 수행 4. break 문이나 switch 문의 끝을 만나면 switch 문의 전체를 빠져나감

일치하는 case 문이 없을 경우에는 default 문을 작성하여 일치하는 값이 없을경우에 수행할 실행문을 작성할 수 있다. default문은 작성을 해도되고, 안해도 되는 구문이다.

그렇다면 switch문의 제약조건은 무엇일까.

💡 1. switch문의 조건식 결과는 정수 또는 문자열이여야 한다. 2. case문의 값은 정수와 상수만 가능하며, 중복될 수 없다.

switch문에 문자열이 가능한 것은 jdk1.7부터 가능하게 되었다. 조건식의 결과가 정수나 문자열임에 따라 조건에 맞는 case문을 찾게 되며, case문은 정수와 상수만 가능하고, 중복될 수 없다.

double squ = (Math.random()*6)+1; // 난수생성을 통한 주사위 범위 구현(1~6)

switch(squ) {
	case 1:
		System.out.println("주사위 1이 나왔습니다.);
	case 2:
		System.out.println("주사위 2가 나왔습니다.);
	case 3:
		System.out.println("주사위 3이 나왔습니다.);
	case 4:
		System.out.println("주사위 4가 나왔습니다.);
	case 5:
		System.out.println("주사위 5가 나왔습니다.);
	case 6:
		System.out.println("주사위 6이 나왔습니다.);
}

squ에 난수 메소드를 사용하여 1부터 6까지의 주사위의 값을 나오게 할 수 있는 변수를 구현하였다. squ 변수를 switch문에 조건식으로 사용하고 나오는 각 수치마다 주사위 값을 출력하도록 하는 실행문을 작성하였다.

if문을 사용해도 다음의 구문은 충분히 구현이 가능하다. 하지만 코드가 복잡해질 뿐더러 가독성도 좋지않다. switch 구문의 제약조건을 만족한다면, switch 구문으로 작성하여 보다 가독성 높은 코딩이 가능하다.

반복문 : while, do-while, for 문

반복문은 어떤 작업이 반복적인 수행을 하도록 할 수 있으며, 반복문의 종류로는 for문과 while, do-whild문이 있다. 반복문은 조건식의 결과가 참이면 문장을 반복적으로 수행하고 거짓이 나올 때까지 이 수행작업은 반복된다. for문은 주로 반복횟수를 알고 있을 때 많이 사용하며, whild문은 반복횟수를 알 수 없을 때 사용된다.

while문

whild문은 if문처럼 조건식과 블럭 {}만으로 이루어져 있다. 하지만 if문과 달리 while문은 조건식이 false를 반환할 때까지 문장을 반복실행한다.

조건식을 먼저 확인하여 조건식이 거짓이면 while문 전체를 벗어나고, 조건식이 true이면 while문을 실행한다. 조건식이 false를 반환할 때까지 계속 반복된다.

int sum = 0;
int num = 1;

while(num<=10) {
		sum += num;
		num++
}

System.out.println(sum); //55

num 변수가 11이 되기 전까지 sum 변수에 값을 더하여 저장하도록 반복문을 작성하였다. num 변수는 1부터 10 차례대로 증감연산자에 의해서 수치가 증가하고, 수치마다 반복문이 실행될 때마다 sum 변수에 num 변수를 합하여 저장하였다. sum을 출력하면 55가 콘솔에 출력될 것이다.

do-while 문

do-while문은 while 문의 변형이다. 기본적인 구조는 while 문과 같지만, 반복문을 먼저 실행한다는 점에서 차이점을 가지고 있다. 한마디로 조건식과 반복문의 순서를 바꾸어 놓은 것으로, 조건식에 관계없이 무조건 반복문을 최소 한 번은 실행하게끔 되어있다.

do {
		// 조건식이 참일 때 수행될 코드를 작성
		// 조건식에 관계없이 최소 한 번은 실행이 된다.
} while(조건식) }
}

for 문

반복횟수를 알고 있을 때 사용하기 적합한 반복문으로, while문보다는 복잡한 구조를 가지고 있지만 이해한다면 while문보다 더 용이하게 사용하게될 반복문이다. 먼저 for문의 구조를 살펴보도록 하자.

for(int i=0; i<=5; i++) { //(초기화; 조건식; 증감식)
	System.out.println("문장");
}

for문은 초기화, 조건식, 증감식 3개의 식을 채워서 사용이 가능하다. 초기화에는 사용할 변수를 선언하면서 초기화하는 식, 조건식은 초기화한 변수를 어느 범위까지 사용할 것인지에 대한 조건을 만든 식, 증감식은 반복문이 끝나면 변수를 얼만큼 증감시킬지를 작성하는 식이다.

위의 조건문에서는 i를 0부터 5까지 하나 씩 숫자를 증가시키면서 반복문을 실행한다. 반복문에는 “문장”이라는 문자열을 한번 출력하도록 작성하였다. 콘솔에서는 6줄의 문장이라는 문구가 출력되어 있을 것이다.

while문에 비해서 코드가 간결하게 작성된 것을 볼 수 있다. 그 만큼 많이 사용되는 반복문이며, 개념적인 이해가 반드시 되어있어야 한다. 이후에 나오는 다중 for문은 다소 로직이 복잡해 이해하는데에 시간이 걸릴 수 있다.

다중 for 문

if문 내에 if문을 넣을 수 있는 것 처럼 for 문 안에도 for문을 넣는 다중 for문 작성이 가능하다. 다중 for문은 보통 별찍기를 통해서 접근하는게 이해할 때 도움이 많이 된다.

/* *****
	 *****
	 *****
	 *****
*/

	for(int i=1; i<=4; i++) {
		for(int k=1; k<=5; k++) {
			System.out.print("*")
		}
		System.out.println();
	}

총 네 줄이고 한 줄에 5개의 별(*)이 있는 문장을 콘솔에 출력해보려고 한다. 다중 for문을 사용하면 쉽게 문장을 출력할 수있다. 안쪽 for문에서는 별을 총 5번 출력하는 반복문이 작성되어있고, 바깥쪽 for문에서는 줄띄움 실행문이 4번 반복되도록 다중 for문이 작성되었다.

이 외에도 바깥 for문과 안쪽 for문에 선언된 변수들을 조건식에 사용하여 상호작용할 수 있는 다중 for문의 작성도 가능하다.

/* *
   **
   ***
   ****
   *****
*/

	for(int i=1; i<=5; i++) {
		for(int j=1; j>=i; j++) {
			System.out.println("*");
		}
		System.out.println();
	}

바깥 for문에 선언된 i 변수를 안쪽 for문에 조건식에 사용하였다. 안쪽 for문에 선언된 j 변수는 i 변수가 조건식에 따라서 수치가 증가함에 따라서 조건식의 반복횟수가 달라지게 된다. 이런 개념을 이용하여 삼각형으로 점찍기를 콘솔에 출력할 수 있다.

이 외에 삼각형을 거꾸로 뒤집거나, 반사시켜서 만들어 보는 등 여러가지 점찍기 구문 작성을 통해 중첩 for문을 보다 자세하게 이해할 수 있다.
