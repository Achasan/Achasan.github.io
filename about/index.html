Memory Repo. 💭

by Achasan 🙋‍♂️

  • 📖 공부한 내용들을 기록하는 블로그입니다.

    • 그날그날 공부한 내용들을 정리하여 기록합니다.

  • Notion과 Tistory를 거쳐 현재의 블로그를 관리하게 되었습니다.

    • Notion은 포스팅만 하기에는 기능이 너무 많습니다. 간단한 기능을 선호하는 저에게는 공부해야할 것도 많고 굳이 배울필요를 못느꼈습니다.
    • Tistory는 블로깅하기 좋은 플랫폼입니다. 만족하면서 썼지만 뭔가 부족했습니다. 테마가 마음에 들지않고, 커스텀하기도 불편했습니다.

  • GitBlog를 사용하면 여러가지 이점을 볼 수 있었습니다.

    • 포스팅을 하면 Github에 commit이 됩니다. 잔디관리도 되고, 블로그에 글도 올라갑니다. 🌱
    • 원하는 기능을 직접 만들거나 라이브러리로 적용할 수 있습니다. (highlight.js, lunr.js 등)
    • 직접 블로그를 만들어 관리하므로 책임감이 상승합니다(?) 👍

  • 이 블로그는 Jekyll을 사용하여 만들어졌습니다. ✨

    • 테마는 jasper2를 기반으로 만들었습니다.
